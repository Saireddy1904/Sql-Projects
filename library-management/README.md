# Library Management System

A comprehensive SQL-based library management system for tracking books, members, borrowing records, and inventory.

## Features

- **Book Management**: Add, update, and manage book inventory
- **Member Management**: Track library members and their information
- **Borrowing System**: Monitor book checkouts and returns
- **Fine Management**: Calculate and track late fees
- **Reports**: Generate various library reports

## Database Structure

### Tables

- `books` - Book inventory and details
- `members` - Library member information
- `borrowing_records` - Track book borrowing transactions
- `fines` - Track overdue fines
- `categories` - Book categories/genres

## Setup

See the SQL scripts in the `scripts/` directory to set up the database.

## Usage

Refer to the queries in the `queries/` directory for common operations.
